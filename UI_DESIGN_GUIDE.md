# UI 设计指南

> 本文件是所有 UI 开发的唯一设计准则。
> Phase 2 每轮开发必须严格遵循本指南，确保全局视觉一致性。
> 风格关键词：**温暖治愈、贴心管家、亲和力强**

## 1. 设计原则
- **温暖亲和**：用暖色调和柔和的视觉元素传递关怀感，让用户感受到被照顾
- **信息克制**：每个页面聚焦核心内容，避免信息过载，保持呼吸感
- **交互自然**：操作反馈即时温和，过渡动画流畅不突兀，如管家轻声回应
- **一致可预期**：所有模块遵循统一的视觉语言，用户不需要重新学习

## 4. 间距系统
- 基础单位：4px
- 间距阶梯：4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 px
- 组件内间距：通常 12-16px
- 组件间间距：通常 16-24px
- 区块间间距：通常 32-48px

## 5. 圆角规范
| 元素 | 圆角 | 说明 |
|------|------|------|
| 按钮 | 12px | 柔和圆角，温和不尖锐 |
| 输入框 | 12px | 与按钮统一 |
| 卡片 | 16px | 大圆角增加亲和感 |
| 对话气泡 | 18px | 更圆润的气泡形态 |
| 头像 | 50% | 圆形 |
| 功能图标容器 | 14px | 图标背景容器 |

## 6. 阴影规范
| 层级 | 阴影值 | 用途 |
|------|--------|------|
| 低 | `0 2px 8px rgba(61, 48, 40, 0.06)` | 卡片、列表项 |
| 中 | `0 4px 16px rgba(61, 48, 40, 0.10)` | 下拉框、悬浮面板 |
| 高 | `0 8px 32px rgba(61, 48, 40, 0.14)` | 弹窗、模态框 |

> 注意：阴影使用暖色调 rgba，避免冷灰色阴影破坏温暖感。

## 7. 核心组件规范

### 7.1 按钮
- **主按钮**：背景 `#E8835A`，文字白色，hover 背景 `#D6744D`，active 背景 `#C4653F`
- **次按钮**：背景 `#FFF9F5`，边框 `#E8835A`，文字 `#E8835A`，hover 背景 `#FFF0E8`
- **柔和按钮**：背景 `#F5EDE6`，文字 `#3D3028`，hover 背景 `#EDE3DA`（用于不需要强调的操作）
- **危险按钮**：背景 `#E06B5E`，文字白色
- **禁用态**：opacity 0.45，cursor not-allowed
- **尺寸**：大 (44px高 / 16px字号) / 中 (36px高 / 14px字号) / 小 (28px高 / 12px字号)
- **图标按钮**：36x36px，圆角 50%，hover 背景 `#F5EDE6`

### 7.2 输入框
- 高度：44px
- 边框：1.5px solid `#F0E6DD`
- 背景：`#FFFFFF`
- 聚焦：边框 `#E8835A`，外发光 `0 0 0 3px rgba(232, 131, 90, 0.12)`
- 错误态：边框 `#E06B5E`，外发光 `0 0 0 3px rgba(224, 107, 94, 0.12)`
- 占位符文字色：`#9B8E82`

### 7.3 对话气泡
- **用户消息**：
  - 背景：渐变 `linear-gradient(135deg, #E8835A, #EE9B78)`
  - 文字：白色
  - 右对齐
  - 圆角：18px 18px 4px 18px（右下角小圆角）
- **管家消息**：
  - 背景：`#FFFFFF`
  - 文字：`#3D3028`
  - 左对齐
  - 圆角：18px 18px 18px 4px（左下角小圆角）
  - 左侧有管家头像（温暖笑脸图标）
- 最大宽度：容器的 75%
- 内间距：12px 16px
- 消息间距：8px
- 时间戳：居中，`#9B8E82`，12px

### 7.4 导航栏
- 位置：底部固定（移动端友好）
- 高度：64px
- 背景：`#FFFFFF`，顶部 1px `#F0E6DD` 边框
- 导航项：图标(24px) + 文字(11px)，间距 4px
- 选中态：图标和文字变为 `#E8835A`，图标上方有 4px 圆形指示点
- 未选中态：图标和文字 `#9B8E82`
- 核心导航项：
  - 🏠 首页（今日概览）
  - 💬 对话（聊天界面）
  - 📊 面板（数据统计）
  - ⚙️ 设置

### 7.5 侧边栏（桌面端展开）
- 宽度：260px（可折叠为 72px 图标栏）
- 背景：`#FFFFFF`
- 左侧功能分组：
  - 常用：对话、今日概览
  - 生活：日程、待办、记账、购物清单
  - 健康：饮食、习惯打卡
  - 工具：天气、新闻、出行、邮件、笔记、读书
  - 智能家居
- 选中项：背景 `#FFF0E8`，文字 `#E8835A`，左侧 3px 圆角条 `#E8835A`

### 7.6 卡片
- 背景：`#FFFFFF`
- 圆角：16px
- 阴影：低阴影
- 内间距：20px
- hover：阴影升级为中阴影，transform translateY(-1px)
- 边框：无（依赖阴影区分层级）

### 7.7 今日概览卡片（主动关怀）
- 特殊设计：背景渐变 `linear-gradient(135deg, #FFF0E8, #FFF9F5)`
- 管家头像 + 问候语（"早上好，今天天气不错~ ☀️"）
- 分区块展示：天气、日程、待办、提醒
- 每个区块有对应的图标和颜色标识

### 7.8 统计图表
- 图表库：Recharts
- 配色：使用主色系（`#E8835A`, `#7BB6A4`, `#F2B84B`, `#8BC0E0`, `#D4A0D9`）
- 圆角柱状图：borderRadius 8px
- 折线图：平滑曲线，线宽 2px，端点圆形 4px

## 8. 响应式断点
| 断点名 | 宽度 | 布局调整 |
|--------|------|----------|
| mobile | < 768px | 单列布局，底部导航，隐藏侧边栏 |
| tablet | 768-1024px | 可折叠侧边栏（图标模式），底部导航隐藏 |
| desktop | > 1024px | 完整侧边栏 + 主内容区 |

## 9. 动画与过渡
- 默认过渡：`all 0.2s ease`
- 页面切换：`fade 0.3s ease-in-out`
- 弹窗出现：`scale(0.95→1) + fade`，`0.25s ease-out`
- 对话消息出现：`translateY(8px→0) + fade`，`0.2s ease-out`
- 加载动画：三个暖色圆点依次跳动（bounce），颜色 `#E8835A`
- 打卡成功：圆形涟漪扩散 + 对勾弹入，绿色 `#7BB6A4`
- 数据刷新：骨架屏（Skeleton），背景色 `#F5EDE6` → `#FFF0E8` 闪烁

## 10. 图标规范
- 风格：Rounded / Outlined（柔和线条，契合温暖风格）
- 图标库：Lucide React（线条风格统一，圆角自然）
- 尺寸：16px (内联) / 20px (按钮内) / 24px (导航) / 32px (功能入口)
- 颜色：跟随文字色或使用功能对应色

## 11. 管家人设视觉
- **管家头像**：温暖橘红色圆形背景，白色笑脸图标
- **问候语风格**：根据时间段变化
  - 早晨（6-11点）："早上好~今天也要元气满满哦！"
  - 中午（11-14点）："午安~记得好好吃午饭呀"
  - 下午（14-18点）："下午好~今天辛苦了"
  - 晚上（18-22点）："晚上好~今天过得怎么样？"
  - 深夜（22-6点）："夜深了，早点休息哦~"
- **语气词**：适当使用"~"、"呀"、"哦"增加亲和力
- **表情/Emoji**：适量使用，增加温暖感（☀️🌙💪✨🎉）
